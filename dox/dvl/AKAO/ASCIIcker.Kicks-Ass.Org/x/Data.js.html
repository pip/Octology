NAME

Form.Validator.Report.Data - no action taken, only data filled

SYNOPSIS

Usage:

  var fv = new Form.Validator(document.f);
  var r  = fv.reporter('Data');
  fv.set('name', 'notBlank', 'Name is required');
  fv.validate();
  var messages = r.messages;

DESCRIPTION

This report object doesn't do anything visibly observable upon validation. All it does is fill in some internal data structures so that test scripts can inspect them later.

Think of this as Form.Validator.Report.AlertAll without the alert(). It's reason for existing is to facilitate testing of this framework.

PROPERTIES

messages

This is an Array that contains a list of onFailure messages for constraints that failed during the call to fv.validate().

METHODS

start

This method clears the contents of this.messages just before validation starts.

run

This method populates this.messages whenever a constraint fails during validation.

finish

This method does nothing.
